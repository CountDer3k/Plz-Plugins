# PlzSuite Collection

## A collection of the Plz plugins for Minecraft.

Please use paper for your Minecraft server for these to function properly.

These plugins are no longer being built to support Spigot. If enough demand, support will be added back, but for now they are paper exclusive.

The Minecraft version supported is after the plugin name, the pluing version comes after the MC version.

Example:

* PlzSuite-1.21.10-1.0

Plugin: PlzSuite
MC Version: 1.21.10
Plugin Version: 1.0


* NoCamiPlz-1.20.1-alpha

Plugin: NoCamiPlz
MC Version: 1.20.1
Plugin Version: alpha
