# PlzSuite

## Description
All the Plz plugins in one convinent place. This contains all the Plz plugins as a single plugin with the ability to enable/disable individual plugins on the fly. This is the only Plz plugin that does not require PlzCore (It's Built-in!). This has all the features from the other plugins so check them out individually to see all features.

## Features:
- BigAnnouncer
- NoBoringPaintings
- NoCamiPlz
- NoSwearPlz
- PlzImproveMC
- PlzPVPThere
- PlzRestInGrave

## Planned Features:
- GUI to enable/disable individual modules (the standalone plugins)
- settings GUI to access all settings of all plugins
- Automatic checker to disable any Plz plugin (jar) on the server to make sure the plugin doesn't load twice & breaks functionality
