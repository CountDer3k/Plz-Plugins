# PlzCore

## Description
This is a Java Plugin library used for all ***Plz*** plugins.

## Features:
- Creating plugin directories for storage
- Creating plugin files
- Moving files buil into plugin into the server's directories
- Add/Remove permission using LuckPerms/Bukkit
- Registering Commands
- Registering Listeners/Events
- Reloading all Plz plugins (or any other plugin that registers itself with PlzCore)

## Commands
<u>-- PlzCore Commands --</u>

- /plzreload - Reloads all the plugins registered (dependent) on PlzCore

## Dependencies
 - LuckPerms (Soft dependency)

## Permissions
### Op by default permissions
- plzcore.admin - Allows the player to use the PlzCore commands
