# PlzImproveMC

## Description
This will include QOL (Quality of Life) updates to Minecraft. This will be an ongoing plugin that will add more features as requested.

## Features:
- Break Spawners with SILK_TOUCH

## Planned Features:
- GUI
- Ability to enable/disable features
