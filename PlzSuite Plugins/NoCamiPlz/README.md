# NoCamiPlz

## Description
Control Abusive Management Interactions (C.A.M.I.) is a plugin to block unlawful distribution of goodies in creative for certain players.
It prevents players from giving away items when in creative. Useful if you have a friend/family member you don't fully trust with all the perks of creative.
This will only affect player who you add the "blocked" permission to.

## Features:
- Prevents players from dropping items in creative (if permission given)
- Prevents players from placing items in chest (any chest) while in creative (if permission given)
- Deletes all player's items on death to prevent them dropping (if permission given)

## Commands
<u>-- NoCamiPlz Commands --</u>

- /plznocami <add | remove> <PLAYER(S)> - Adds/Removes a player's ability to drop items or place items in inventories, inventory cleared on death.

  ***PLAYERS must be separated by a space if adding more than one at a time***

## Dependencies
 - PlzCore

## Permissions
### Op by default permissions
- plznocami.admin - Allows the player to use the PlzNoCami commands
- plznocami.blocked - Disables a user from dropping items or placing items in inventories, inventory cleared on death.
